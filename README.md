# faprotax2_database_manuscript
This repo stores the code used to analyze/visualize data for FAPROTAX2-db manuscript.
PLEASE NOTE:
- This repo does not contain code used to compile, clean, and build the reference phylogeny for FAPROTAX2-db.
- This is not meant to be a place to download all components of FAPROTAX2. Such code, as well as the FAPROTAX2 software, are archived and downloadable at www.loucalab.com/archive/FAPROTAX2.
- Figures generated by this workflow underwent successive manual editing. For instance, points in Figure 1 (FAPROTAXv1_vs_v2_counts.pdf) were annotated to label specific functions with high and low representation in FAPROTAX2-db. Additionally, the annotated phylogeny and corresponding legend in Figure 4 (FAPROTAX2_annotated_tree.pdf) were manually enlarged to improve readability.

## Requirements
- R version 4.4.1
- Packages listed in R scripts

## To reproduce figures
Clone this repository, ensure all necessary R packages are installed, then run the "workflow.sh" bash script from the command line.

## Data availability
The data used for this project come from the FAPROTAX2-db phenotypic database, a database on the metabolic functions of 13,574
prokaryotic species. For more information on FAPROTAX2 and its underlying database, please visit www.loucalab.com/archive/FAPROTAX2.
