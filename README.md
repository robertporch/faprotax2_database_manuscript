# faprotax2_database_manuscript
Code used to analyze/visualize data for FAPROTAX2-db manuscript

## Requirements
- R version 4.4.1
- Packages listed in R scripts

## To reproduce figures
Clone this repository, ensure all necessary R packages are installed, then run the "workflow.sh" bash script from the command line.

## Data availability
The data used for this project come from the FAPROTAX2-db phenotypic database, a database on the metabolic functions of 13,574
prokaryotic species. For more information, please visit www.loucalab.com/archive/FAPROTAX2.
