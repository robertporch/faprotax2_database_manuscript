# faprotax2_database_manuscript
This repo stores the code used to analyze/visualize data for FAPROTAX2-db manuscript.
PLEASE NOTE that this repo does not contain code used to compile, clean, and build the reference phylogeny for FAPROTAX2-db.
This is also not meant to be a place to download all components of FAPROTAX2.
Such code, as well as the FAPROTAX2 software, are archived and downloadable at www.loucalab.com/archive/FAPROTAX2.

## Requirements
- R version 4.4.1
- Packages listed in R scripts

## To reproduce figures
Clone this repository, ensure all necessary R packages are installed, then run the "workflow.sh" bash script from the command line.

## Data availability
The data used for this project come from the FAPROTAX2-db phenotypic database, a database on the metabolic functions of 13,574
prokaryotic species. For more information on FAPROTAX2 and its underlying database, please visit www.loucalab.com/archive/FAPROTAX2.
